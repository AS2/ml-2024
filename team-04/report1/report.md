## Изменения в проекте с 30.09 по 06.10

1) На UI добавлен флажок для работы с ML. Если флажок включен, разрешены действия в программе, связанные с данным проектом. Иначе действия недоступны.
                                                                                                                                                                                                                                                                                                                                                                                                                                  
2) Добавлена функциональность отрисовки желаемой области работ (прямоугольник). Для построения области необходимо нажать на ЛКМ - тогда поставится угловая точка желаемой области. При движении курсора мыши автоматически строится область, которая формируется за счет выбранной угловой точки и курсора мыши по диагонали от выбранной угловой точки. При еще одном нажатии ЛКМ вторая угловая точка (и желаемая область работ) фиксируется в программе. При нажатии Esc желаемая область работ стирается.
См. видео "workzone.mp4"

3) Добавлена функциональность записи в файл обучающих данных для модели машинного обучения. Если в программе существуют все 8 якорей и желаемая область работ, то происходит запись в файл. Файл состоит из колонок в следующем порядке:
Широта первой угловой точки - долгота первой угловой точки - широта второй угловой точки - долгота второй угловой точки - широта якоря №1 - долгота якоря №1 - ... - широта якоря №8 - долгота якоря №8
См. файл "ML_data_example.dat"
